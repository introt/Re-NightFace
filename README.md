# Re-NightFace
A clean room re-implementation of [NightFace Pebble watchface](https://apps.getpebble.com/en_US/application/56f9e26f69581dad78000025) with a twist: instead of showing the time when tapped, it shows the time on startup and then unloads.  
  
[Available on the Rebble Appstore](https://apps.rebble.io/en_US/application/5b0cf87bf38588a252000e30)! [archive.org mirror of old Pebble Appstore](http://web.archive.org/web/20180529071234/https://apps.getpebble.com/en_US/application/5b0cf87bf38588a252000e30).
